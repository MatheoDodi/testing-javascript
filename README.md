## Getting Started

To get a local copy of this repo up and running follow these simple steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/MatthewDodi/testing-javascript
```

2. Install NPM packages

```sh
npm install
```

3. Run the Development Server

```sh
npm start
```
